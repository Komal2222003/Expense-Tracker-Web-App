# Expense Tracker Web Application

## Description
A simple Expense Tracker application that helps users manage and track their daily expenses.

## Features
- Add new expenses
- View expense list
- Calculate total expenses
- Delete expenses
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
Expense-Tracker-Web-App/
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run
1. Download the project files.
2. Open `index.html` in any web browser.
3. Start adding and tracking your expenses.
